# CodeAlpha - Stock Portfolio Tracker 📈

This project is developed as part of the Python Programming Internship at CodeAlpha by Kaunain Hussain.

## 🔍 Project Overview

The Stock Portfolio Tracker is a simple Python program that calculates the total investment value of a user's stock holdings based on predefined prices.

### 👨‍💻 Features
- Accepts user input for stock names and quantities.
- Uses a hardcoded dictionary for stock prices (e.g., AAPL, TSLA).
- Calculates and displays the total investment value.
- Optionally saves the result to a `.txt` file.

### 🧠 Concepts Used
- Dictionary
- User Input / Output
- Loops & Conditions
- Basic File Handling
